Seguridad en Azure

  • Proteja su empresa de amenazas avanzadas en todas las cargas de trabajo en la nube híbrida

  • App Configuration

    Almacenamiento de parámetros rápido y escalable para la configuración de aplicaciones

  • Application Gateway

    Cree front-ends web seguros, escalables y de alta disponibilidad en Azure

  • Azure Active Directory

    Sincronice los directorios locales y habilite el inicio de sesión único

  • Azure Active Directory Domain Services

    Unir máquinas virtuales de Azure a un dominio sin controladores de dominio

  • Security Center

    Proteja sus entornos de nube híbrida y multinube

  • Administración de la superficie expuesta a ataques externos de Defender

    Proteja la experiencia digital mediante la detección de todos los recursos expuestos a Internet con la Administración de superficie expuesta a ataques externos (EASM) de Microsoft Defender

  • Azure Bastion

    Servicio totalmente administrado que ayuda a proteger el acceso remoto a las máquinas virtuales

  • Azure DDoS Protection

    Proteja sus recursos de Azure frente a ataques por denegación de servicio distribuido (DDoS)

  • Azure Dedicated HSM

    Administre los módulos de seguridad de hardware que utiliza en la nube

  • Azure Firewall

    Proteja sus recursos de Azure Virtual Network con seguridad de red nativa en la nube

  • Azure Firewall Manager

    Administración central de rutas y directivas de seguridad de red para perímetros definidos por software y distribuidos globalmente

  • Azure Front Door

    Red CDN moderna en la nube que ofrece experiencias optimizadas a los usuarios en cualquier lugar

  • Azure Information Protection

    Mejore la protección de la información confidencial, en todo momento y en cualquier parte

  • Azure Sentinel

    Utilice la funcionalidad SIEM nativa en la nube y análisis de seguridad inteligentes para mejorar la protección de su empresa

  • Key Vault

    Proteja y mantenga el control de las claves y otros secretos

  • VPN Gateway

    Establecer conectividad segura entre entornos locales

  • Firewall de aplicaciones web

    Un servicio de firewall de aplicaciones web (WAF) nativo en la nube que proporciona una protección muy eficaz para las aplicaciones web

  • Microsoft Azure Attestation

    Solución unificada para comprobar de forma remota la confiabilidad de una plataforma y la integridad de los archivos binarios que se ejecutan en ella

  • Azure confidential ledger

    Almacén de datos no estructurados resistente a alteraciones hospedado en entornos de ejecución de confianza (TEE) y con el respaldo de una evidencia que se pueden comprobar criptográficamente
